 ## Run Locally

Clone the project

```bash
  git clone https://github.com/Faizan7/Activity-Tracker.git
```

Go to the project directory

```bash
  cd activity-tracker
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
